# Online Font Preview Tool

This repository shares a useful online typography preview resource
that helps designers and creators test fonts with custom text
without installing them on their system.

## Why this exists
I am sharing this based on personal experience.
It saves time, keeps devices lightweight,
and improves the font selection workflow.
